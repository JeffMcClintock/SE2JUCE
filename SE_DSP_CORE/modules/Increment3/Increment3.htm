Increment3


Steps up or down through a list of values.  Usually connected to a 'Patch Mem - List3'.

Pins

  • Choice Connect to the value you want to increment. e.g. Patch Mem 'Choice' pin.
  • Item List Connect this to the  associated list-of-values.  e.g. Patch Mem 'Item List' pin.
  • Increment Connect to a button via a 'Bool to Float'.
  • Decrement Connect to a button via a 'Bool to Float'.
  • Wrap Incrementing last value will jump back to first value (wrap-arround).
