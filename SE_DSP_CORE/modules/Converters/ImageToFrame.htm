Image to Frame

Animated images in synthedit are usually controlled by a normalised value, between zero and one. This module converts that normalised value to a frame number. For example you may have an animation containing the numbers one through five, this module helps you extract the index of any frame.
 This is useful when the Image links to a list of options, like using pictures of several waveshapes to select the oscillator waveform.

Pins

  • Animation Position
  • Frame Count
  • Frame Number
