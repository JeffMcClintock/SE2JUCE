GMPI Protoype SDK

2005-04-10



This code is a prototype SDK for the GMPI API.  As the GMPI API is 
developed, this code will change dramatically.  This code is not official.  

This SDK is intended to be as cross-platform as possible.  If you find
some aspect of it that is not, please let us know.  Bug reports and patches
are very welcome.
