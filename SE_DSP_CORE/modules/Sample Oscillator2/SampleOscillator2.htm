There are 3 module for Sample Playback.

  • Sample Loader2 - loads samples.

  • Sample Oscillator2 - plays samples.

  • Sample Loader2 Gui - is for hooking up patch and bank selection.

See prefab...Examples->Sample Player2.  It shows how to hook a working sample oscillator.

Note: When using Sample playback with the Keyboard2 module, always set the Keyboard2 Allocation Mode to "Poly (Hard)". This prevents clicks when playing the same key repeatedly.

- Sample Exclusive Hihat - Mutes Closed Hihat (note 42) when Open Hihat ( note 46) plays and vica-versa.
