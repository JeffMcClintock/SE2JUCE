System Command

Allows you to trigger a Operating System command on any file.  Handy for opening external help files, or your homepage in an external web browser.

Pins

  • Trigger - Usually connect to a Sub Control button image.
  • Command - An integer representing what action to take.
  • Command List - List of supported actions. "edit, explore, find, open, print, properties".
  • Filename - The file to Open or whatever.
